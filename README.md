### GIT remote branches

## check remote
 `git remote -v`

## add upstream

 `git remote add upstream git@github.com:kizouker/git-workshops.git`


## add origin

 `git remote add origin git@github.com:kizouker/git-workshops.git`

### Fix ssh keys

### Create Branch
 `git branch -M main`


 ## Push 
Pushes the main branch to the origin
 `git push -u origin main`
